Документация Weather-2-Fly



        Приложение предназначено для определения погоды в определяемом по координатам регионе и вычисления наилучшего времени для полетов на квадрокоптере типа "Дрон".
        Вычисление благоприятного времени для осуществления полетов производится на основании данных, рекомендуемых документацией дрона.
        Вывод информации производится в виде коэффициента от 0 до 1 с градацией 0.25.
